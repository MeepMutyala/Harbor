## publishing

for GitHub Pages:

1. repository Settings → Pages
2. Source: "Deploy from a branch"
3. select branch and `/whitepaper` folder
4. save

## local preview

```bash
cd whitepaper
bundle install
bundle exec jekyll serve
```

open http://localhost:4000
